Referencia Rápida

Comandos esenciales de PowerShell para uso diario

Fundamentos

Ayuda y Documentacion
Get-Help
Muestra ayuda sobre un cmdlet
Get-Help -Examples
Muestra ejemplos de uso
Get-Help -Detailed
Muestra ayuda detallada
Update-Help
Actualiza la ayuda local
Descubrimiento de Comandos
Get-Command
Lista todos los comandos disponibles
Get-Command -Verb
Filtra por verbo (Get, Set, New, Remove...)
Get-Command -Noun *
Filtra por sustantivo (Process, Service, Item...)
Get-Alias
Lista todos los alias disponibles

Sistema de Archivos

Navegacion y Listado
Get-ChildItem
Lista archivos y carpetas (equivalente a dir/ls)
Get-ChildItem -Path C:\carpeta
Lista contenido de una ruta especifica
Get-ChildItem -Recurse
Lista recursivamente incluyendo subcarpetas
Get-ChildItem -Filter "*.txt"
Filtra por extension de archivo
Get-ChildItem -Hidden
Muestra archivos ocultos
Set-Location C:\carpeta
Cambia de directorio (cd)
Get-Location
Muestra directorio actual (pwd)
Gestión de Archivos
New-Item -ItemType File "archivo.txt"
Crea un nuevo archivo
New-Item -ItemType Directory "carpeta"
Crea una nueva carpeta (mkdir)
Copy-Item "origen" -Destination "destino"
Copia archivos o carpetas (cp)
Move-Item "origen" -Destination "destino"
Mueve archivos o carpetas (mv)
Remove-Item "archivo" -Force
Elimina archivos/carpetas (rm, -Recurse para carpetas)
Rename-Item "archivo" "nuevonombre"
Renombra archivos o carpetas (ren)
Test-Path "ruta"
Verifica si existe una ruta (True/False)

Procesos y Sistema

Gestion de Procesos
Get-Process
Lista todos los procesos en ejecucion
Get-Process -Name "nombre"
Obtiene proceso especifico por nombre
Get-Process | Where-Object {$_.CPU -gt 10}
Procesos que usan mas de 10 seg de CPU
Stop-Process -Name "nombre"
Detiene un proceso por nombre
Stop-Process -Id 1234
Detiene proceso por ID
Start-Process "notepad.exe"
Inicia un proceso
Start-Process "archivo.txt" -Verb RunAs
Ejecuta como administrador
Gestion de Servicios
Get-Service
Lista todos los servicios
Get-Service -Name "nombre"
Obtiene servicio especifico
Get-Service | Where-Object {$_.Status -eq 'Running'}
Solo servicios en ejecucion
Start-Service -Name "nombre"
Inicia un servicio
Stop-Service -Name "nombre"
Detiene un servicio
Restart-Service -Name "nombre"
Reinicia un servicio
Set-Service -Name "nombre" -StartupType Automatic
Cambia tipo de inicio (Automatic/Manual/Disabled)

Variables y Objetos

Declaracion de Variables
$variable = "valor"
Crea una variable (siempre con $)
$numero = 123
Variable numerica
$arreglo = @(1, 2, 3)
Crea un arreglo
$hash = @{clave = "valor"}
Crea una tabla hash
Get-Variable
Lista todas las variables
Remove-Variable -Name "variable"
Elimina una variable
Inspeccion de Objetos
$objeto | Get-Member
Muestra propiedades y metodos de un objeto
$objeto | Select-Object -Property *
Muestra todas las propiedades
$objeto.Propiedad
Accede a una propiedad especifica
$objeto.GetType()
Muestra el tipo de dato
$null -eq $variable
Verifica si una variable es null

Pipeline y Filtros

Filtrado y Seleccion
Where-Object {$_.Propiedad -eq "valor"}
Filtra objetos por condicion (abreviatura: Where)
Select-Object -Property prop1, prop2
Selecciona propiedades especificas
Sort-Object -Property Nombre
Ordena por propiedad (ascendente)
Sort-Object -Property Valor -Descending
Ordena de forma descendente
Group-Object -Property Tipo
Agrupa objetos por propiedad
Measure-Object -Property Valor -Sum -Average
Calcula suma, promedio, etc.
Operaciones de Lista
Select-Object -First 10
Toma los primeros 10 elementos
Select-Object -Last 5
Toma los ultimos 5 elementos
Select-Object -Unique
Elimina duplicados
ForEach-Object {$_.Propiedad}
Ejecuta para cada elemento (abreviatura: ForEach)
Where-Object -First 1
Toma el primer elemento que cumple la condicion

Red y Conectividad

Redes
Test-Connection -ComputerName "host"
Verifica conectividad (equivalente a ping)
Test-NetConnection -ComputerName "host"
Test de conectividad detallado (como ping + traceroute)
Get-NetIPAddress
Muestra configuraciones de IP
Get-NetAdapter
Lista adaptadores de red
Resolve-DnsName "dominio"
Resuelve DNS (equivalente a nslookup)
Remoting
Enter-PSSession -ComputerName "equipo"
Inicia sesion remota interactiva
Invoke-Command -ComputerName "equipo" -ScriptBlock {comando}
Ejecuta comando en equipo remoto
New-PSSession -ComputerName "equipo"
Crea una sesion persistente
Start-Job -ScriptBlock {comando}
Inicia trabajo en segundo plano
Get-Job | Receive-Job
Obtiene resultados de trabajos

Scripting y Control

Condicionales
if ($condición) { codigo }
Condicion simple
if ($a -eq $b) { } else { }
Condicion con else
if ($a -gt 10 -and $b -lt 20) { }
AND logico
if ($a -eq "x" -or $a -eq "y") { }
OR logico
switch ($valor) { "x" { } "y" { } }
Estructura switch
Bucles
for ($i = 0; $i -lt 10; $i++) { }
Bucle for clasico
ForEach ($item in $coleccion) { }
Bucle foreach
while ($condicion) { }
Bucle while
do { } while ($condicion)
Bucle do-while
1..10 | ForEach-Object { $_ * 2 }
Rango con foreach

Operadores Comparativos

Operadores Numericos
-eq
Igual a (equals)
-ne
Diferente a (not equals)
-gt
Mayor que (greater than)
-ge
Mayor o igual que
-lt
Menor que (less than)
-le
Menor o igual que
Operadores de Cadenas
-like "*.txt"
Coincide con patron (wildcard)
-notlike "*patron*"
No coincide con patron
-match "^abc"
Coincide con expresion regular
-contains "valor"
Contiene valor (para colecciones)
-in (1,2,3)
El valor esta en la coleccion
