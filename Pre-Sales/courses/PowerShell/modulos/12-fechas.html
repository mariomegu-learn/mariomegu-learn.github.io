Módulo 12: Fechas y Formato

Obtener Fecha y Hora

# Obtener fecha y hora actual $fecha = Get-Date Write-Host $fecha # Propiedades comunes $fecha.Year $fecha.Month $fecha.Day $fecha.Hour $fecha.Minute $fecha.Second # Fechas especiales Get-Date -Year 2024 -Month 12 -Day 25 # Navidad Get-Date -DayOfWeek Monday # Próximo lunes (Get-Date).AddDays(7) # Una semana desde hoy (Get-Date).AddHours(-2) # Hace 2 horas

Formatear Fechas

Formato con -Format

# Formatos comunes Get-Date -Format "yyyy-MM-dd" # 2024-01-15 Get-Date -Format "dd/MM/yyyy" # 15/01/2024 Get-Date -Format "HH:mm:ss" # 14:30:45 Get-Date -Format "dddd, dd 'de' MMMM" # Lunes, 15 de Enero # Formato completo Get-Date -Format "dddd, dd 'de' 'de' yyyy 'a las' HH:mm:ss" # Formatos estándar Get-Date -Format "f" # Fecha y hora cortas Get-Date -Format "F" # Fecha y hora largas Get-Date -Format "d" # Fecha corta Get-Date -Format "D" # Fecha larga Get-Date -Format "t" # Hora corta Get-Date -Format "T" # Hora larga

Características de formato

Código Descripción Ejemplo
yyyy Año de 4 dígitos 2024
MM Mes (01-12) 01
dd Día (01-31) 15
HH Hora (00-23) 14
hh Hora (01-12) 02
mm Minutos (00-59) 30
ss Segundos (00-59) 45

Operaciones con Fechas

Sumar y restar tiempo

# Sumar tiempo (Get-Date).AddDays(30) # 30 días desde hoy (Get-Date).AddMonths(6) # 6 meses desde hoy (Get-Date).AddYears(1) # 1 año desde hoy (Get-Date).AddHours(2.5) # 2.5 horas desde hoy (Get-Date).AddMinutes(45) # 45 minutos desde hoy # Restar tiempo (usar negativo) (Get-Date).AddDays(-7) # Hace 7 días (Get-Date).AddMonths(-3) # Hace 3 meses # Diferencia entre fechas $fecha1 = Get-Date $fecha2 = (Get-Date).AddDays(10) $diferencia = $fecha2 - $fecha1 $diferencia.Days # Días de diferencia $diferencia.Hours # Horas de diferencia $diferencia.TotalHours # Horas totales (incluye fracción)

Calcular edad

$fechaNacimiento = Get-Date "1990-05-15" $edad = (Get-Date) - $fechaNacimiento Write-Host "Edad en días: $($edad.Days)" Write-Host "Edad en años: $(($edad.Days / 365).ToString('0'))"

Validar Fechas

# Verificar si es fecha válida $fecha = "2024-02-30" if ($fecha -as [datetime]) { Write-Host "Fecha válida" } else { Write-Host "Fecha inválida" } # Usar TryParse $fecha = "2024-13-45" $resultado = [datetime]::TryParse($fecha, [ref]$null) if ($resultado) { Write-Host "Fecha válida" } else { Write-Host "Fecha inválida" } # Verificar rango $fecha = Get-Date if ($fecha -ge (Get-Date "2024-01-01") -and $fecha -le (Get-Date "2024-12-31")) { Write-Host "Fecha en 2024" }

Usar Fechas en Nombres de Archivos

# Nombre de archivo con timestamp $timestamp = Get-Date -Format "yyyyMMdd_HHmmss" $nombreArchivo = "reporte_$timestamp.txt" # Crear archivo con nombre temporal $archivo = "C:\Logs\log_$(Get-Date -Format 'yyyy-MM-dd').log" # Carpeta por fecha $carpeta = "C:\Backups\$(Get-Date -Format 'yyyy')\$(Get-Date -Format 'MM')\$(Get-Date -Format 'dd')" # Nombre ISO estándar (ordenable) $nombreISO = "backup_$(Get-Date -Format 'yyyyMMddTHHmmss').zip"

Zonas Horarias

# Ver todas las zonas horarias Get-TimeZone -ListAvailable # Zona horaria actual Get-TimeZone # Cambiar zona horaria # Set-TimeZone -Id "Pacific Standard Time" # Convertir hora entre zonas $horaLocal = Get-Date $zonaUTC = Get-TimeZone -Id "UTC" $horaUTC = [System.TimeZoneInfo]::ConvertTime($horaLocal, (Get-TimeZone), $zonaUTC) # Hora UTC Get-Date -UFormat "%Y-%m-%d %H:%M:%S UTC"

Ejercicio Práctico

Práctica: Trabajando con Fechas
  1. Obtén la fecha actual en formato "dd/MM/yyyy HH:mm:ss"
  2. Calcula cuántos días faltan para fin de año
  3. Obtén la fecha de hace 30 días y sumale 6 meses
  4. Calcula tu edad en años, meses y días
  5. Crea un nombre de archivo con timestamp: "backup_YYYYMMDD_HHMMSS.zip"
  6. Verifica si una fecha ingresada por el usuario es válida
  7. Genera un reporte con la fecha en formato ISO 8601
  8. Convierte la hora local a UTC
  9. Obtén el primer y último día del mes actual
  10. Calcula los días hábiles (excluyendo fines de semana) entre dos fechas

Mejores Prácticas

  • Usa formato ISO 8601: yyyy-MM-dd para nombres de archivo (ordenables)
  • Usa UTC para almacenamiento: Convierte a local solo para mostrar
  • Valida entradas de fecha: Siempre verifica que sean fechas válidas
  • Usa propiedades de TimeSpan: Para cálculos de diferencia
  • Considera zonas horarias: En aplicaciones multi-región
