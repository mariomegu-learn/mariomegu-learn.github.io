Módulo 05: Estructuras de Control

Condicionales: If, Else If, Else

Las estructuras condicionales permiten ejecutar código solo si se cumple una condición.

# Estructura básica if $edad = 17 if ($edad -ge 18) { Write-Host "Eres mayor de edad" } # if-else if ($edad -ge 18) { Write-Host "Eres mayor de edad" } else { Write-Host "Eres menor de edad" } # if-else if-else $nota = 85 if ($nota -ge 90) { Write-Host "Excelente" } elseif ($nota -ge 80) { Write-Host "Muy bien" } elseif ($nota -ge 70) { Write-Host "Bien" } else { Write-Host "Necesitas mejorar" }
Consejo: Puedes anidar condicionales, pero intenta mantenerlos simples para mejor legibilidad.

Switch

Switch es una alternativa a múltiples else if cuando comparas un valor contra varias opciones.

# Switch básico $dia = "Lunes" switch ($dia) { "Lunes" { Write-Host "Comienza la semana" } "Miércoles" { Write-Host "Medio día" } "Viernes" { Write-Host "Casi fin de semana" } "Sábado", "Domingo" { Write-Host "Fin de semana" } default { Write-Host "Día de semana" } } # Switch con wildcard $archivo = "reporte.pdf" switch -Wildcard ($archivo) { "*.txt" { Write-Host "Archivo de texto" } "*.pdf" { Write-Host "Documento PDF" } "*.jpg", "*.png" { Write-Host "Imagen" } default { Write-Host "Otro tipo de archivo" } }
# Switch con regex $correo = "usuario@empresa.com" switch -Regex ($correo) { "@gmail\.com$" { Write-Host "Gmail" } "@outlook\.com$" { Write-Host "Outlook" } "@.*\.com$" { Write-Host "Correo comercial" } "@.*\.edu$" { Write-Host "Correo educativo" } default { Write-Host "Otro correo" } }

Bucles: For

El bucle For ejecuta un bloque de código un número específico de veces.

# For básico for ($i = 0; $i -lt 5; $i++) { Write-Host "Iteración número: $i" } # Contar hacia atrás for ($i = 5; $i -gt 0; $i--) { Write-Host "Contando: $i" } # Incremento diferente for ($i = 0; $i -le 10; $i += 2) { Write-Host "Número par: $i" } # Recorrer array $frutas = @("Manzana", "Banana", "Naranja") for ($i = 0; $i -lt $frutas.Count; $i++) { Write-Host "Fruta $i: $($frutas[$i])" }

Bucles: Foreach

Foreach es ideal para recorrer colecciones de elementos.

# Foreach básico $frutas = @("Manzana", "Banana", "Naranja", "Uva") foreach ($fruta in $frutas) { Write-Host "Fruta: $fruta" } # Foreach con índice $colores = @("Rojo", "Verde", "Azul") for ($i = 0; $i -lt $colores.Count; $i++) { Write-Host "Índice $i: $($colores[$i])" } # Foreach con objetos $users = @( @{Nombre="Carlos"; Edad=30}, @{Nombre="Ana"; Edad=25}, @{Nombre="Luis"; Edad=35} ) foreach ($user in $users) { Write-Host "Usuario: $($user.Nombre), Edad: $($user.Edad)" }
# Foreach-Object en pipeline 1..10 | ForEach-Object { $cuadrado = $_ * $_ Write-Host "$_ al cuadrado es $cuadrado" } # Foreach-Object con expresión $frutas | ForEach-Object { $_.ToUpper() }

Bucles: While

While ejecuta un bloque de código mientras una condición sea verdadera.

# While básico $contador = 0 while ($contador -lt 5) { Write-Host "Contador: $contador" $contador++ } # While con condición compleja $numero = 100 while ($numero -gt 10) { Write-Host "Número: $numero" $numero /= 2 } # Leer hasta que el usuario ingrese algo $entrada = "" while ([string]::IsNullOrWhiteSpace($entrada)) { $entrada = Read-Host "Ingresa tu nombre" } Write-Host "Hola, $entrada"
Cuidado: Asegúrate de que la condición eventualmente sea falsa para evitar bucles infinitos.

Bucles: Do-While y Do-Until

Estos bucles ejecutan el código al menos una vez antes de verificar la condición.

# Do-While (verifica al final) $contador = 0 do { Write-Host "Contador: $contador" $contador++ } while ($contador -lt 5) # Do-Until (ejecuta hasta que la condición sea verdadera) $numero = 10 do { Write-Host "Número: $numero" $numero-- } until ($numero -eq 0) # Validación de entrada $password = "" do { $password = Read-Host "Ingresa contraseña (mínimo 8 caracteres)" } until ($password.Length -ge 8) Write-Host "Contraseña aceptada"

Break y Continue

Break sale del bucle inmediatamente. Continue salta a la siguiente iteración.

# Break - salir del bucle for ($i = 1; $i -le 10; $i++) { if ($i -eq 5) { Write-Host "Llegué a 5, saliendo" break } Write-Host "Número: $i" } # Continue - saltar iteración for ($i = 1; $i -le 5; $i++) { if ($i -eq 3) { Write-Host "Saltando 3" continue } Write-Host "Número: $i" } # Buscar un elemento $frutas = @("Manzana", "Banana", "Naranja", "Uva") foreach ($fruta in $frutas) { if ($fruta -eq "Naranja") { Write-Host "Encontré Naranja" break } }

Ejercicio Práctico

Práctica: Control de Flujo
  1. Crea un programa que verifique si un número es positivo, negativo o cero usando if-else
  2. Usa switch para mostrar el nombre del mes dado su número (1=Enero, 2=Febrero, etc.)
  3. Usa un bucle for para imprimir los números del 1 al 100, pero multiplica por 2
  4. Crea un array de 5 nombres y usa foreach para imprimir cada uno con un mensaje
  5. Usa while para crear un contador regresivo desde 10 hasta 0
  6. Implementa una validación de contraseña que pida hasta que tenga al menos 8 caracteres
  7. Usa break para salir de un bucle cuando encuentres un número específico en un array
  8. Usa continue para imprimir solo números pares del 1 al 20

Mejores Prácticas

  • Usa foreach para colecciones: Es más legible que for cuando no necesitas el índice
  • Evita bucles infinitos: Siempre asegúrate que la condición pueda ser falsa
  • Mantén los condicionales simples: Si son muy complejos, considera refactorizar
  • Usa switch para múltiples opciones: Es más limpio que muchos else if
  • Comenta tu lógica: Especialmente cuando uses condiciones complejas
