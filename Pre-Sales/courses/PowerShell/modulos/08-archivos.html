Módulo 08: Archivos y Directorios

Crear Archivos y Directorios

Crear directorios

# Crear un directorio simple New-Item -ItemType Directory -Path "C:\MiCarpeta" # Crear directorios anidados New-Item -ItemType Directory -Path "C:\Carpeta1\Carpeta2\Carpeta3" -Force # Usar alias mkdir "C:\NuevaCarpeta" New-Item "C:\OtraCarpeta" -ItemType Directory

Crear archivos

# Crear archivo vacío New-Item -ItemType File -Path "C:\archivo.txt" # Crear archivo con contenido New-Item -ItemType File -Path "C:\archivo.txt" -Value "Contenido inicial" # Crear archivo y escribir Set-Content -Path "C:\archivo.txt" -Value "Contenido" # Usar alias New-Item "archivo.txt" -ItemType File "Contenido" | Out-File "archivo.txt"
Consejo: Usa -Force para sobrescribir archivos existentes o crear directorios anidados.

Leer Archivos

Leer todo el contenido

# Leer todo el archivo $contenido = Get-Content -Path "C:\archivo.txt" # Leer como un solo string $contenido = Get-Content -Path "C:\archivo.txt" -Raw # Leer líneas específicas $lineas = Get-Content -Path "C:\archivo.txt" $primeraLinea = $lineas[0] $ultimasLineas = $lineas[-5..-1]

Leer con codificación

# Especificar codificación Get-Content -Path "archivo.txt" -Encoding UTF8 Get-Content -Path "archivo.txt" -Encoding Default Get-Content -Path "archivo.txt" -Encoding Unicode # Leer solo las primeras líneas Get-Content -Path "archivo.txt" -TotalCount 10 # Leer solo las últimas líneas Get-Content -Path "archivo.txt" -Tail 10

Escribir en Archivos

Escritura básica

# Sobrescribir contenido Set-Content -Path "archivo.txt" -Value "Nuevo contenido" # Agregar al final Add-Content -Path "archivo.txt" -Value "Nueva línea" # Escribir múltiples líneas $lineas = @("Línea 1", "Línea 2", "Línea 3") Set-Content -Path "archivo.txt" -Value $lineas

Usando Out-File

# Escribir con Out-File Get-Process | Out-File -FilePath "procesos.txt" # Agregar en lugar de sobrescribir Get-Service | Out-File -FilePath "servicios.txt" -Append # Especificar codificación "Contenido" | Out-File -FilePath "archivo.txt" -Encoding UTF8 # Especificar ancho de salida Get-Process | Out-File "salida.txt" -Width 200

Buscar y Listar Archivos

Listar archivos

# Listar todo en un directorio Get-ChildItem -Path "C:\MiCarpeta" # Solo archivos Get-ChildItem -Path "C:\MiCarpeta" -File # Solo directorios Get-ChildItem -Path "C:\MiCarpeta" -Directory # Listar recursivamente Get-ChildItem -Path "C:\MiCarpeta" -Recurse

Buscar por patrón

# Buscar archivos .txt Get-ChildItem -Path "C:\MiCarpeta" -Filter "*.txt" # Buscar archivos .log o .txt Get-ChildItem -Path "C:\MiCarpeta" -Filter "*.log" Get-ChildItem -Path "C:\MiCarpeta" -Filter "*.txt" # Buscar recursivamente por extensión Get-ChildItem -Path "C:\" -Filter "*.ps1" -Recurse -ErrorAction SilentlyContinue # Buscar archivos que empiezan con algo Get-ChildItem -Path "C:\MiCarpeta" -Filter "reporte*.pdf"

Filtrar archivos

# Archivos más grandes que 1MB Get-ChildItem -Path "C:\MiCarpeta" | Where-Object {$_.Length -gt 1MB} # Archivos modificados en los últimos 7 días Get-ChildItem | Where-Object {$_.LastWriteTime -gt (Get-Date).AddDays(-7)} # Archivos creados antes de cierta fecha Get-ChildItem | Where-Object {$_.CreationTime -lt (Get-Date).AddMonths(-1)} # Ocultar archivos Get-ChildItem -Path "C:\MiCarpeta" -Hidden

Modificar Archivos

Copiar, Mover, Renombrar

# Copiar archivo Copy-Item -Path "archivo.txt" -Destination "archivo_copia.txt" Copy-Item -Path "C:\origen\archivo.txt" -Destination "C:\destino\" # Mover archivo Move-Item -Path "archivo.txt" -Destination "carpeta\archivo.txt" Move-Item -Path "archivo.txt" -Destination "nuevo_nombre.txt" # Renombrar archivo Rename-Item -Path "archivo.txt" -NewName "nuevo_archivo.txt" # Usar alias cp archivo.txt copia.txt mv archivo.txt carpeta\ rn archivo.txt nuevo.txt

Eliminar archivos

# Eliminar archivo Remove-Item -Path "archivo.txt" # Eliminar sin confirmar Remove-Item -Path "archivo.txt" -Force # Eliminar archivos por patrón Remove-Item -Path "*.tmp" -Force # Eliminar recursivamente Remove-Item -Path "carpeta" -Recurse -Force # Eliminar archivos más viejos que X días Get-ChildItem -Path "C:\Logs" | Where-Object {$_.LastWriteTime -lt (Get-Date).AddDays(-30)} | Remove-Item

Información de Archivos

# Obtener información de archivo $archivo = Get-Item -Path "archivo.txt" # Propiedades comunes $archivo.Name $archivo.FullName $archivo.Length $archivo.CreationTime $archivo.LastWriteTime $archivo.LastAccessTime $archivo.Extension # Verificar si existe Test-Path -Path "archivo.txt" Test-Path -Path "C:\carpeta" -PathType Container Test-Path -Path "archivo.txt" -PathType Leaf # Obtener ruta Split-Path -Path "C:\carpeta\archivo.txt" # Directorio Split-Path -Path "C:\carpeta\archivo.txt" -Leaf # Nombre del archivo Split-Path -Path "C:\carpeta\archivo.txt" -Parent # Padre

Trabajar con CSV

# Leer CSV $datos = Import-Csv -Path "datos.csv" $datos | ForEach-Object { Write-Host $_.Nombre } # Crear CSV $usuarios = @( [PSCustomObject]@{Nombre="Carlos"; Edad=30; Ciudad="Madrid"}, [PSCustomObject]@{Nombre="Ana"; Edad=25; Ciudad="Barcelona"}, [PSCustomObject]@{Nombre="Luis"; Edad=35; Ciudad="Valencia"} ) $usuarios | Export-Csv -Path "usuarios.csv" -NoTypeInformation # Agregar a CSV existente $usuarioNuevo = [PSCustomObject]@{Nombre="Maria"; Edad=28; Ciudad="Sevilla"} $usuarioNuevo | Export-Csv -Path "usuarios.csv" -Append -NoTypeInformation

Ejercicio Práctico

Práctica: Manipulación de Archivos
  1. Crea una carpeta llamada "Prueba" con subcarpetas "Sub1" y "Sub2"
  2. Crea 3 archivos de texto con contenido diferente en cada subcarpeta
  3. Lee el contenido de todos los archivos .txt y muestra el nombre junto con el contenido
  4. Busca todos los archivos .ps1 en tu carpeta de documentos
  5. Crea un script que copie todos los archivos .log de una carpeta a otra
  6. Implementa una función que elimine archivos más viejos de 30 días
  7. Crea un archivo CSV con información de productos (nombre, precio, stock)
  8. Convierte un CSV a JSON usando ConvertTo-Json
  9. Crea un script que recorra recursivamente una carpeta y cuente archivos por extensión

Mejores Prácticas

  • Usa rutas completas: Evita problemas con el directorio actual
  • Verifica existencia antes de operar: Usa Test-Path
  • Maneja errores: Usa try-catch para operaciones de archivo
  • Usa codificación UTF8: Para compatibilidad internacional
  • No uses nombres con espacios: O cita las rutas: "C:\Mi Carpeta\archivo.txt"
  • Usa -WhatIf para pruebas: Remove-Item -Path "*.txt" -WhatIf
