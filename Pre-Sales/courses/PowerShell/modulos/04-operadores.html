Módulo 04: Operadores y Expresiones

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas básicas.

Operador Descripción Ejemplo
+ Suma $a + $b
- Resta $a - $b
* Multiplicación $a * $b
/ División $a / $b
% Módulo (resto) $a % $b
++ Incremento $a++
-- Decremento $a--
# Ejemplos de operadores aritméticos $a = 10 $b = 3 Write-Host "Suma: $($a + $b)" # 13 Write-Host "Resta: $($a - $b)" # 7 Write-Host "Multiplicación: $($a * $b)" # 30 Write-Host "División: $($a / $b)" # 3.333... Write-Host "Módulo: $($a % $b)" # 1 # Incremento y decremento $c = 5 $c++ Write-Host "Incrementado: $c" # 6 $c-- Write-Host "Decrementado: $c" # 5

Operadores de Comparación

Los operadores de comparación se utilizan para comparar valores. En PowerShell, los operadores de comparación comienzan con -.

Operador Descripción Ejemplo
-eq Equal (igual a) $a -eq $b
-ne Not equal (no igual) $a -ne $b
-gt Greater than (mayor que) $a -gt $b
-lt Less than (menor que) $a -lt $b
-ge Greater than or equal $a -ge $b
-le Less than or equal $a -le $b
# Ejemplos de comparaciones $edad = 25 Write-Host ($edad -eq 25) # True Write-Host ($edad -gt 18) # True Write-Host ($edad -lt 30) # True Write-Host ($edad -ge 25) # True Write-Host ($edad -ne 30) # True # Comparación de strings $nombre = "Carlos" Write-Host ($nombre -eq "Carlos") # True Write-Host ($nombre -ne "Ana") # True # Comparación case-insensitive (por defecto) Write-Host ($nombre -eq "carlos") # True
Case-sensitive: Usa -ceq, -cgt, etc. para comparaciones sensibles a mayúsculas/minúsculas.

Operadores Lógicos

Los operadores lógicos se utilizan para combinar expresiones booleanas.

Operador Descripción Ejemplo
-and AND lógico $a -and $b
-or OR lógico $a -or $b
-not NOT lógico -not $a
! NOT lógico (alternativo) !$a
# Ejemplos de operadores lógicos $edad = 25 $tieneLicencia = $true # AND - ambas condiciones deben ser verdaderas if ($edad -ge 18 -and $tieneLicencia) { Write-Host "Puede conducir" } # OR - al menos una condición debe ser verdadera $esFestivo = $false if ($esFestivo -or $esSabado) { Write-Host "Hoy no hay que trabajar" } # NOT - invierte el valor $activo = $true if (-not $activo) { Write-Host "Usuario inactivo" } else { Write-Host "Usuario activo" # Este se ejecuta }

Operadores de Strings

Concatenación

# Concatenación con + $nombre = "Carlos" $apellido = "García" $nombreCompleto = $nombre + " " + $apellido Write-Host $nombreCompleto # Carlos García # Interpolación de strings (recomendado) $nombreCompleto = "$nombre $apellido" Write-Host $nombreCompleto # Carlos García # Interpolación con expresiones $edad = 25 Write-Host "Tengo $edad años, en 5 años tendré $($edad + 5)"

Operadores de comparación de strings

$texto = "PowerShell" # Contains Write-Host ($texto -like "*Shell*") # True Write-Host ($texto -like "Power*") # True Write-Host ($texto -like "*Shell") # True # StartsWith / EndsWith Write-Host ($texto -like "Power*") # True Write-Host ($texto -like "*Shell") # True # Regex Write-Host ($texto -match "Shell") # True Write-Host ($texto -match "python") # False # Reemplazo $nuevo = $texto -replace "Power", "Microsoft" Write-Host $nuevo # MicrosoftShell

Operadores de Asignación

Operador Descripción Equivalente
= Asignación $a = 5
+= Sumar y asignar $a += 3
-= Restar y asignar $a -= 2
*= Multiplicar y asignar $a *= 2
/= Dividir y asignar $a /= 2
# Ejemplos de asignación $contador = 0 $contador += 1 # 1 $contador += 5 # 6 $contador -= 2 # 4 $contador *= 2 # 8 $contador /= 2 # 4 # Con strings $mensaje = "Hola" $mensaje += " Mundo" # "Hola Mundo" # Con arrays $lista = 1, 2, 3 $lista += 4 # 1, 2, 3, 4

Operadores Especiales

Operador -in (pertenece a)

# Verificar si un valor está en una lista $color = "azul" $coloresPermitidos = "rojo", "verde", "azul" if ($color -in $coloresPermitidos) { Write-Host "Color válido" } # También funciona inverso $coloresPermitidos -contains $color # True

Operador -join (unir)

# Unir array en string $palabras = "Hola", "Mundo", "PowerShell" $resultado = $palabras -join " " Write-Host $resultado # "Hola Mundo PowerShell" # Unir sin separador $numero = 1, 2, 3, 4, 5 $id = $numero -join "" Write-Host $id # "12345"

Operador -split (dividir)

# Dividir string en array $texto = "manzana,banana,naranja,uva" $frutas = $texto -split "," Write-Host $frutas[0] # "manzana" # Dividir por espacio $oracion = "PowerShell es genial" $palabras = $oracion -split " " Write-Host $palabras.Count # 3

Ejercicio Práctico

Práctica: Usando Operadores
  1. Calcula el área de un rectángulo: crea variables para base y altura, luego multiplica
  2. Crea un programa que verifique si un número es par usando el operador módulo (%)
  3. Usa operadores lógicos para verificar si una edad está entre 18 y 65
  4. Crea un string con tu nombre y apellido, luego usa -split para separarlos
  5. Usa -join para unir un array de palabras en una oración
  6. Verifica si un color está en una lista de colores permitidos usando -in
  7. Crea un contador que incremente de 0 a 10 usando +=
  8. Usa interpolación de strings para mostrar: "En 10 años tendré [edad + 10] años"

Mejores Prácticas

  • Usa interpolación de strings: Es más legible que concatenar con +
  • Usa paréntesis para claridad: ($a + $b) * $c en lugar de $a + $b * $c
  • Recuerda la precedencia: * y / tienen prioridad sobre + y -
  • Usa operadores case-sensitive cuando sea necesario: Con la 'c' (eq, ne, gt, etc.)
  • Prueba tus comparaciones: Los strings se comportan diferente a los números
