Módulo 01: Introducción a PowerShell

¿Qué es PowerShell?

PowerShell es un shell de línea de comandos y lenguaje de scripting desarrollado por Microsoft. Combina la potencia de la línea de comandos con la flexibilidad de un lenguaje de scripting para automatizar tareas de administración de sistemas.

Originalmente llamado Windows PowerShell, fue lanzado en 2006 como parte de Windows. En 2016, Microsoft lo hizo de código abierto y multiplataforma, pasando a llamarse simplemente PowerShell.

Dato importante: PowerShell está construido sobre .NET, lo que significa que puede acceder a clases .NET, COM objects y WMI directamente desde la consola.

Historia de PowerShell

PowerShell 1.0 (2006)

Lanzado con Windows Vista y Windows Server 2008. Introdujo conceptos fundamentales como cmdlets, pipeline de objetos y scripting básico.

PowerShell 2.0 (2009)

Añadido a Windows 7 y Windows Server 2008 R2. Introdujo remoting, jobs en background, transacciones y mejores capacidades de scripting.

PowerShell 3.0-5.1 (2012-2016)

Mejoras continuas con Windows 8, Windows 10 y Windows Server 2012/2016. Introducción de DSC (Desired State Configuration) y mejor integración con Azure.

PowerShell Core 6+ (2016-presente)

PowerShell de código abierto y multiplataforma que corre sobre .NET Core. Funciona en Windows, Linux y macOS.

PowerShell vs CMD vs Otras Shells

Característica CMD PowerShell
Base MS-DOS legacy .NET Framework/Core
Salida Texto plano Objetos .NET
Capacidades Básicas Avanzadas
Scripting Muy limitado Completo
Multiplataforma No Yes (PowerShell 6+)
Importante: PowerShell no es compatible hacia atrás con CMD. Los comandos de CMD no funcionan en PowerShell sin modificaciones.

Instalación de PowerShell

En Windows 10/11

PowerShell viene preinstalado. Para verificar la versión:

$PSVersionTable.PSVersion

Instalar PowerShell 7+ (Recomendado)

  1. Descargar desde: github.com/PowerShell/PowerShell
  2. Instalador MSI para Windows
  3. O usar Windows Package Manager: winget install Microsoft.PowerShell

En Linux

# Ubuntu/Debian sudo apt update sudo apt install powershell # RHEL/CentOS sudo dnf install powershell

Primeros Comandos

Comandos básicos para empezar

Get-Host
Muestra información sobre el host de PowerShell actual
Get-Command
Lista todos los cmdlets disponibles
Get-Help Get-Command
Muestra ayuda detallada sobre un cmdlet específico
Clear-Host
Limpia la pantalla de la consola (alias: clc)
Get-Location
Muestra el directorio actual (alias: pwd)

Conceptos Clave

Cmdlets

Los cmdlets son comandos nativos de PowerShell con el formato Verbo-Nombre. Ejemplos: Get-ChildItem, Set-Location, New-Item.

Pipeline (|)

El pipeline permite pasar la salida de un cmdlet como entrada a otro. A diferencia de CMD, PowerShell pasa objetos, no texto.

# Ejemplo de pipeline Get-Process | Where-Object {$_.CPU -gt 100} | Sort-Object CPU -Descending

Aliases

PowerShell tiene aliases para comandos comunes. Por ejemplo: ls funciona como Get-ChildItem, cd como Set-Location.

Consejo: Usa Get-Alias para ver todos los aliases disponibles.

Ejercicio Práctico

Práctica: Explorando PowerShell
  1. Abre PowerShell en tu sistema (Win + X, luego selecciona "Windows PowerShell" o "Terminal")
  2. Ejecuta $PSVersionTable para ver la versión instalada
  3. Ejecuta Get-Command -CommandType Cmdlet para ver cuántos cmdlets tienes disponibles
  4. Usa Get-Help Get-Process para leer la ayuda del cmdlet Get-Process
  5. Ejecuta Get-Process | Select-Object -First 5 para ver los primeros 5 procesos
  6. Prueba Get-Service | Where-Object {$_.Status -eq "Running"} | Select-Object -First 10
  7. Usa Get-Alias | Where-Object {$_.Definition -like "*Child*"} para encontrar aliases relacionados

Mejores Prácticas Iniciales

  • Usa Get-Help frecuentemente: PowerShell tiene documentación integrada muy completa
  • Aprovecha el autocompletado: Presiona Tab para completar comandos automáticamente
  • Usa aliases para velocidad: gci en lugar de Get-ChildItem cuando escribas rápido
  • Explora el pipeline: Combina cmdlets para crear consultas poderosas
  • Conoce los verbos estándar: Get, Set, New, Remove, Update, Start, Stop, etc.
