¿Por qué PowerShell?

PowerShell es el lenguaje de scripting nativo de Windows y el estándar para la administración de sistemas Microsoft.

  • Automatización de tareas repetitivas
  • Administración de sistemas Windows y Azure
  • Gestión de servicios en la nube
  • Integración con Active Directory y Exchange
