Clasificación de Ciberataques

Explore las diversas categorías de ciberataques, comprenda sus subtipos y los objetivos que persiguen. Esta guía exhaustiva le ayudará a identificar y protegerse de las amenazas digitales más comunes y sofisticadas.

bug_report

1. Malware

expand_more
  • Virus: Infectar archivos y propagarse a otros sistemas.
  • Gusano: Replicarse y propagarse a través de redes sin intervención humana.
  • Troyano: Disfrazarse de software legítimo para obtener acceso no autorizado.
  • Ransomware: Cifrar archivos y exigir un rescate para su liberación.
  • Spyware: Recopilar información de un dispositivo sin el consentimiento del usuario.
groups

2. Ingeniería Social

expand_more
  • Phishing: Engañar a las víctimas para que revelen información sensible a través de correos electrónicos fraudulentos.
  • Vishing: Utilizar llamadas telefónicas para manipular a las víctimas.
  • Smishing: Emplear mensajes de texto (SMS) para el engaño.
  • Baiting: Dejar dispositivos infectados (como USBs) para que las víctimas los encuentren y usen.
  • Pretexting: Crear un escenario falso para obtener información.
psychology

3. Ataques con IA

expand_more
  • Deepfakes: Crear videos o audios falsos y convincentes para difamar o engañar.
  • Generación de Mensajes Falsos: Crear correos electrónicos de phishing o noticias falsas altamente personalizados y creíbles.
  • Envenenamiento de Datos: Manipular los datos de entrenamiento de un modelo de IA para corromper sus resultados.
lan

4. Ataques de Red

expand_more
  • Denegación de Servicio (DoS/DDoS): Inundar un servicio con tráfico para hacerlo inaccesible.
  • Man-in-the-Middle (MitM): Interceptar y alterar la comunicación entre dos partes.
  • Spoofing: Falsificar la identidad (IP, correo electrónico) para engañar a los sistemas o usuarios.
code

5. Ataques a Aplicaciones Web

expand_more
  • Inyección SQL: Insertar código SQL malicioso para manipular bases de datos.
  • Cross-Site Scripting (XSS): Inyectar scripts maliciosos en sitios web para ejecutarlos en los navegadores de los usuarios.
  • Falsificación de Peticiones (CSRF): Engañar a un usuario para que realice acciones no deseadas en un sitio web en el que está autenticado.
sensor_door

6. Ataques Físicos

expand_more
  • Robo de dispositivos: Sustraer portátiles, teléfonos o servidores para acceder a los datos.
  • Acceso no autorizado: Ingresar a instalaciones para instalar malware o robar información.
  • Dumpster Diving: Rebuscar en la basura en busca de documentos o dispositivos con información sensible.
